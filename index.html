Remote Secuirty Camera

A motion tracker than can turn any webcam into a remote motion tracker. The motion tracker can be control via SMS, currently with START and STOP commands. The user is also notified when motion is detected via SMS and a video copy of the motion is stored in the user's Dropbox.The application was intentially created to work on the Raspberry Pi, however it can work on any linux based computer that has OpenCV installed. Various filters are used to limit motion detection to only large movements, lowering the possibility of false alerts. To view the source code, click here

  • Type: Independent Project
  • Date: December 2015
  • Technologies: Python, Flask, OpenCV, Twilio, Dropbox

Memory Box

An Android application that allows young onset individuals in early stages of Dementia to log and retrieve memory notes. The phone application will record the voice initiated memory log made by the user and stores it into a cloud based server. A web portal was also created to show statistical feedback of user’s memory progression and habits developed over time. The application was developed for DementiaHack 2015, the source code for the application can be found here.

  • Type: Independent Project
  • Date: November 2015
  • Technologies: Android, Python, MySQL, Flask

Indoor Navigation

Designed a real-time navigation system that can guide visitors and students to any room within the University of Guelph campus. A combination of Dead Reckoning and Dijkstra's algorithm were used to plot the most efficient path to the area of interest relative to the user's position. Image processing was used on customized floor plans to identify key locations within buildings. The application was designed for Engineering Design III, and the report for the application can be found here. Currently the project is of purely design and has not been implemented.

  • Type: Academic Project
  • Date: April 2015
  • Technologies: Sensors, life Cycle Analysis, Dead reckoning, Dijkstra's Algorithm, Image Processing

Gaze Controlled Mouse

Computer mouse cursor that is controlled from where you look on the screen. This project was done for our engineering capstone project, applying our knowledge of hardware, software design and image processing. The gaze controlled mouse cursor uses a simple gaze estimation algorithm and pupil detection algorithm to estimate the user's approximate gaze on screen and moves the mouse cursor accordingly. The source code can be found here.

  • Type: Academic Project
  • Date: April 2016
  • Technologies: C++, OpenCV, Distributed Collaboration, Algorithms, Arduino, Nvidia TK1
